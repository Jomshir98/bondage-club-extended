Collected data

BCX Collects some data for internal statistics. All data is kept first-party and only used for statistics and development decision purposes.
Data is collected exclusively through BC's in-game "Beep" mechanism, meaning any data outside of BC (such as IP addresses) isn't and cannot be collected.
Full list of collected data is:
  • Version of BCX and BC
  • If using normal or development version of BCX
  • Which link was used to load BCX (or that it was loaded from another mod)
  • UserAgent header of your browser
  • If in a public room (and space of the room) or if in a private room - obtained automatically through the Beep data
  • MemberNumber of the account sending beep - obtained through the Beep data and used exclusively to prevent duplicate entries
These are collected as part of the "Version check" mechanism. You can check the code responsible for this HERE
